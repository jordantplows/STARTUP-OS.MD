# Investor Department

Manages all investor relations, fundraising preparation, and due diligence.

## Agents

- **target-investors** — Research agent for identifying and scoring target VCs
- **data-room** — Assembles investor due diligence on demand
- **narrative** — Maintains canonical fundraising story
- **term-sheet** — Analyzes term sheets and flags founder-unfavorable clauses

## Purpose

Keeps fundraising materials always current. When traction improves, deck updates. When model changes, deck updates. Identifies best-fit investors and builds warm intro paths. Prepares founder for every stage of fundraising process.

## Coordination

- Reads all company.os state for deck and data room generation
- Writes investor pipeline to `company.os.investor`
- Coordinates with legal/ on term sheet analysis
- Coordinates with cfo/ on financial projections
- Coordinates with cmo/ on narrative alignment

## Instructions

Define the technology stack and tools for the startup, including:

1. **Development Stack**
   - Programming languages
   - Frameworks and libraries
   - Development tools and IDEs

2. **Infrastructure**
   - Cloud provider (AWS, GCP, Azure)
   - Hosting and deployment
   - CI/CD pipeline

3. **Data & Storage**
   - Database systems
   - Data warehousing
   - File storage

4. **Communication & Collaboration**
   - Team communication (Slack, Discord)
   - Project management (Linear, Jira)
   - Documentation (Notion, Confluence)

5. **Business Tools**
   - CRM system
   - Email platform
   - Analytics tools
   - Payment processing

6. **Security Tools**
   - Authentication service
   - Secrets management
   - Monitoring and logging
